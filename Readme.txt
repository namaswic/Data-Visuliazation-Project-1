To run this project you need the following things:
R
Rstudio
Shiny(library of R)
Web Browser

Steps:
Note: All datasets and files downloaded should be kept in the same folder
1)Open either server.R or ui.r contained in the downloaded folder in Rstudio.
2)On the top right corner of the editor, you see the Run App button. Press the button to run the application. There are three options where you can run the app: External Brower, window and pane, choose whatever you want to view in.
3)You app should run and perform all good now
